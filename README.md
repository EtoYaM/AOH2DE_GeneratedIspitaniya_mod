## Мод для игры [Age of History 2: Definitive Edition](https://store.steampowered.com/app/3381680/Age_of_History_2_Definitive_Edition/)
Добавляет автогенерируемые испытания на основе существующих в игре формируемых цивилизаций. Может иногда не срабатывать из-за ошибок в проработке цивилизаций. Проверялся на версии игры 2.01.

Мод предназначен для [SourceAPI AOH2:DE](https://github.com/EtoYaM/SourceAPI_AOH2DE/tree/master).

см. [туториал по установке](https://github.com/EtoYaM/SourceAPI_AOH2DE/blob/master/README.md)
